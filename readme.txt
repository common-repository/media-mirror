== Description ==

Plugin for integration with Media Mirror affiliate panel. Imports your affiliates links, create and sorts tables. Enable POPUP.

= How do I register for a Media Mirror account? =
You can sign up at www.mediamirror.net

= A plugin for Media Mirror affiliates = 
This plugin is for users who run Media Mirror on self-hosted WordPress platforms. You'll need to register a Media Mirror channel and get your channel token/api-key to get 100% out of this plugin. Media Mirror connects and downloads your program relations to your platform by using the Media Mirror API. It also downloads and installs image attachments to your Media Gallery. You can sign up for a new account at www.mediamirror.net.

You can read more about how this works in the FAQ section. 

If you have any question regarding this plugin, please email us at info@mediamirror.net or visit our website www.mediamirror.net

= Major features in Media Mirror Plugin include =
* Import of affiliate programs from Media Mirrors API
* Table creation for casino listings
* Popup settings for the Media Mirror javascript popup
* Available in Sweden, Norway, Denmark, UK/US and Finland

== Frequently Asked Questions ==

= Do I need to register a Media Mirror account before using this plugin? =
No, but you will not be able to download your affiliate links or program data without an API-key.

= Does this plugin connect with any 3rd party services? =
Yes, if you enable syncronization by entering your channel ID and channel token under Media Mirror settings. This plugin connects to Media Mirror by calling our JSON api (https://cdn.mediamirror.net/api/table.json). 

= Does this plugin send any anonymous data to 3rd party services? =
No, we do not send any information to any 3rd party services. We do send your channel id and channel token to our own API (https://cdn.mediamirror.net/api/table.json) to be able to download your affiliate links and programs.

= Where do I find my channel id and channel/api token? =
You'll find them under your channel page in our Backoffice under the section "Channel Info". 

= What informations is downloaded from my Media Mirror account? =
This plugin downloads a complate list of your affiliate programs together with the information needed to display a full affilaite table (program name, your unique affiliate links & bonus/coupon offers). It also downloads and installs image attachments (program logo) to your Media Gallery.

== Installation ==

1. Upload the entire `mediamirror` folder to the `/wp-content/plugins/` directory or download it from Plugins in WordPress Dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find the 'Media Mirror' tab in your WordPress admin panel.
